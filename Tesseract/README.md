# Tesseract — Research Scratch Space

This directory is the default ephemeral workspace for the Research Assistant. Session directories are created automatically when research begins and deleted when findings are stored or discarded.

The scratch space path is configurable via `config/config.json` → `researchScratchSpace`.

This is where we conduct and process research. The expectation is that items here will either be published to the Reliquary for long term storage, merged with other work such as Foundry content or Nexus projects, our discarded as one time items.
